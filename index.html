Modern Centering: Flex & Grid

Flexbox Centering

Centered!
.container {
  display: flex;
  justify-content: center;
  align-items: center;
}

Grid Centering

Centered!
.container {
  display: grid;
  place-items: center;
}

New School: Block Element Centering

align-content on Block Elements

Centered!

  align-content: center;
  height: 200px; /* needs explicit height */
}
