View Transitions API

üé¨ View Transitions in Action

Seamless State Changes

This slide transitioned smoothly from the previous one

Using document.startViewTransition()

‚ú® Magic of View Transitions

Browser-Native Animations

Each slide transition is handled by the browser

Smooth, performant, and accessible

What's Happening Behind the Scenes

JavaScript API

// Start a view transition
const transition = document.startViewTransition(() => {
  // Update the DOM
  targetSlide.classList.add('active');
  currentSlide.classList.remove('active');
});

// Optional: customize the transition
transition.ready.then(() => {
  // Apply custom animations to pseudo-elements
  document.documentElement.animate({
    clipPath: ['circle(0)', 'circle(100%)']
  }, {
    duration: 500,
    pseudoElement: '::view-transition-new(root)'
  });
});

CSS Pseudo-Elements

/* Browser automatically creates these */
::view-transition
::view-transition-group(*)
::view-transition-image-pair(*)
::view-transition-old(*)
::view-transition-new(*)

/* You can style them */
::view-transition-old(root) {
  animation: slide-out 0.3s ease-out;
}

::view-transition-new(root) {
  animation: slide-in 0.3s ease-in;
}
